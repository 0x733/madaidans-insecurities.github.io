Firefox and Chromium Security

Chromium is far more secure than Firefox. Firefox lacks many security features that Chromium has such as site isolation, a hardened memory allocator (mozjemalloc is just a few security features tacked on to jemalloc and is not similar to a hardened memory allocator), Control-Flow Integrity and many more. Firefox's current security features such as its browser sandbox are a lot worse than Chromium's. Excluding the issue of site isolation, only the Firefox sandbox on Windows is similar to Chromium's. The others are very insecure and the Linux sandbox can hardly be called a sandbox at all as there are many trivial escapes such as the X11 server (this is also a 5 year old, critical issue).

Just look at what security experts such as thegrugq and Daniel Micay have to say about Firefox.
